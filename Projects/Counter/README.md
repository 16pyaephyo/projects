# Counter App

## Project Overview
This is a simple Counter application built using **HTML, CSS, and JavaScript**.  
The main goal of this project is to practice **JavaScript logic**, **DOM manipulation**, and **event handling**.

The app allows the user to increase, decrease, and reset a number by clicking buttons.

---

## Features
- Increase counter value
- Decrease counter value
- Reset counter to zero
- Simple and clean user interface
- Instant update without page refresh

---

## Technologies Used
- HTML – structure of the app
- CSS – layout, colors, and button styling
- JavaScript – logic and interaction

---

## How It Works
- The number is displayed inside an HTML element
- Buttons listen for `click` events
- JavaScript updates the value and re-renders it on the screen
- No external libraries are used

---

## What I Learned
- How to select elements using `querySelector`
- How to use `addEventListener`
- How JavaScript variables change state
- How to update text content dynamically
- How to keep logic simple and readable

---

## Challenges & Solutions
**Challenge:**  
At first, I didn’t understand why the number was not updating.

**Solution:**  
I learned that I must update the DOM every time the value changes.
