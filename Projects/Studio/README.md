# Wedding Photo Studio Website

## Project Overview
This is a **responsive website design** for a Wedding Photo Studio.  
The project focuses on **layout design, responsiveness, and UI/UX**, using **HTML and CSS**.

This project does not include JavaScript logic and focuses on **visual presentation**.

---

## Features
- Responsive layout for desktop, tablet, and mobile
- Clean and elegant design
- Image gallery section
- Service and contact sections
- Smooth layout flow

---

## Technologies Used
- HTML – semantic structure
- CSS – Flexbox, Grid, media queries
- Responsive design techniques

---

## Design Approach
- Mobile-first mindset
- Flexible layouts using percentages
- Media queries for different screen sizes
- Visual balance and spacing

---

## What I Learned
- Responsive design principles
- Using Flexbox and Grid together
- Structuring large layouts
- Improving visual hierarchy
- Designing for real clients

---

## Challenges & Solutions
**Challenge:**  
Layout breaking on small screens.

**Solution:**  
I adjusted container widths and used media queries properly.
