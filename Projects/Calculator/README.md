# Calculator App

## Project Overview
This project is a basic Calculator built using **HTML, CSS, and JavaScript**.  
It performs common arithmetic operations and focuses on **JavaScript logic and event handling**.

---

## Features
- Addition, subtraction, multiplication, division
- Clear / reset functionality
- Button-based input
- Real-time calculation display
- Clean and simple layout

---

## Technologies Used
- HTML – buttons and display layout
- CSS – grid layout and visual design
- JavaScript – calculation logic

---

## How It Works
- Each button sends input to JavaScript
- Numbers and operators are stored as strings
- JavaScript evaluates the expression
- The result is shown on the display

---

## What I Learned
- Handling multiple button events
- Managing user input step by step
- Preventing invalid calculations
- Separating logic and UI
- Writing cleaner JavaScript functions

---

## Challenges & Solutions
**Challenge:**  
Handling operators and numbers together was confusing.

**Solution:**  
I broke the logic into small steps and tested each part separately.
