# Form Validation Project

## Project Overview
This project focuses on **validating user input** using **JavaScript**.  
The form checks user data before submission and gives feedback messages.

---

## Features
- Required field validation
- Email format validation
- Password length check
- Error and success messages
- Prevents invalid form submission

---

## Technologies Used
- HTML – form structure
- CSS – error and success styles
- JavaScript – validation logic

---

## How It Works
- User submits the form
- JavaScript checks each input
- Errors are shown if data is invalid
- Form submits only when all checks pass

---

## What I Learned
- How forms work in HTML
- Preventing default form behavior
- Writing conditional logic
- Showing error messages dynamically
- Improving user experience

---

## Challenges & Solutions
**Challenge:**  
The form submitted even with empty inputs.

**Solution:**  
I learned to use `event.preventDefault()` correctly.
