# To-Do List App

## Project Overview
This is a To-Do List application built using **HTML, CSS, and JavaScript**.  
The goal of this project is to practice **CRUD operations**, **DOM manipulation**, and **user interaction**.

---

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Dynamic list rendering
- Simple and user-friendly UI

---

## Technologies Used
- HTML – input field and list structure
- CSS – layout and task styling
- JavaScript – task logic and events

---

## How It Works
- User enters a task and clicks add
- JavaScript creates a new list item
- Tasks are added and removed dynamically
- Completed tasks are visually styled

---

## What I Learned
- Creating elements dynamically
- Using arrays to store data
- Handling multiple event listeners
- Managing UI updates
- Writing reusable functions

---

## Challenges & Solutions
**Challenge:**  
Tasks disappeared when refreshing the page.

**Solution:**  
I understood the need for storage and learned about `localStorage` (future improvement).

---

## Possible Improvements
- Save tasks using localStorage
- Edit existing tasks
- Add task categories or filters

## Conclusion
This project helped me understand **real-world app behavior** and improved my confidence in JavaScript.
